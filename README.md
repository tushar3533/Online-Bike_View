# Bike Portal

## Features
- Bike listing
- Search by brand & budget
- EMI Calculator
- Offer page

## Setup

1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt

3. Setup database:
   Run database_backup.sql in PostgreSQL

4. Run app:
   python app.py
